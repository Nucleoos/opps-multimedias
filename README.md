opps-videos
===========

Video Application for Opps


Requirements
-------------

Opps


Configuration
-------------

Include opps.videos on your django settings

```python
INSTALLED_APPS += (
    'opps.videos'
)
```

Create DB tables:
```
python manage.py syndb
```
